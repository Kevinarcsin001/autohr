## Artifacts

### API Endpoints

#### POST /api/auth/register
- **Purpose:** 注册新用户（首位自动 admin + 创建默认 team）
- **Location:** backend/app/api/auth.py:118
- **Request Format:** {email: EmailStr, password: str, name: str}
- **Response Format:** AuthResponse {user: UserOut, tokens: TokenPair} + Set-Cookie: autohr_refresh

#### POST /api/auth/login
- **Purpose:** 邮箱密码登录（CITEXT 大小写不敏感）
- **Location:** backend/app/api/auth.py:139
- **Request Format:** {email: EmailStr, password: str}
- **Response Format:** AuthResponse + Set-Cookie: autohr_refresh

#### POST /api/auth/refresh
- **Purpose:** 用 refresh token 换新 access（不轮换 refresh）
- **Location:** backend/app/api/auth.py:155
- **Request Format:** cookie: autohr_refresh 或 body {refresh_token: str}
- **Response Format:** {access_token: str, token_type: 'Bearer'}

#### POST /api/auth/logout
- **Purpose:** 清除 refresh cookie
- **Location:** backend/app/api/auth.py:177
- **Request Format:** 无 body
- **Response Format:** 204 No Content

#### POST /api/auth/invite
- **Purpose:** admin 发起团队邀请（48h 过期，一次性 token）
- **Location:** backend/app/api/auth.py:185
- **Request Format:** {email: EmailStr, role: 'admin'|'member', name?: str}
- **Response Format:** InviteOut {id, email, role, invite_token, expires_at}

#### GET /api/auth/invites
- **Purpose:** 列出当前 team 所有邀请（admin only）
- **Location:** backend/app/api/auth.py:209
- **Request Format:** 无
- **Response Format:** InviteOut[]

#### POST /api/auth/accept-invite
- **Purpose:** 通过邀请链接注册新用户并加入团队
- **Location:** backend/app/api/auth.py:235
- **Request Format:** {invite_token: str, name: str, password: str}
- **Response Format:** AuthResponse

#### GET /api/auth/me
- **Purpose:** 获取当前登录用户
- **Location:** backend/app/api/auth.py:252
- **Request Format:** Authorization: Bearer <access>
- **Response Format:** UserOut
